# Home Work - Менеджер Событий/Планировщик

## Описание

Приложение, разработанное на **C++Builder VCL**, которое функционирует как менеджер событий или простой планировщик. Основная цель приложения — отслеживать события или задачи с привязкой к датам и отображать количество дней, оставшихся до каждого события.


## Ключевые Особенности

* **Отслеживание Событий:** Просмотр списка событий/задач с их названиями и датами.
* **Обратный Отсчет:** Автоматический расчет и отображение количества дней, оставшихся до каждого события.
* **Управление Данными:** Возможность добавления, редактирования и удаления элементов.
    * **Редактирование Названия:** Специальная форма для изменения текстового описания события.
    * **Выбор Даты:** Использование календаря для выбора точной даты события.
    * **Быстрый Выбор Даты:** Вспомогательная форма для установки даты через заданное количество недель.
* **Сохранение Данных:** Сохранение данных в локальные файлы (`.dat`).
* **Системный Трей:** Поддержка минимизации в область уведомлений (системный трей) с возможностью восстановления окна.
* **Один Экземпляр:** При попытке повторного запуска, вторая копия уведомит первую и закроется, а первая копия восстановит свое окно и получит фокус.

Проект разработан с использованием Embarcadero RAD Studio.
